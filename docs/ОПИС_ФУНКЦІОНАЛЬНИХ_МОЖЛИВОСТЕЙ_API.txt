ОПИС ФУНКЦІОНАЛЬНИХ МОЖЛИВОСТЕЙ API

Instagram надає можливість швидко отримати інформацію, яка знаходиться в профілі
за допомогою API.

Подавши запит можна отримати:

	> список підписок користувача (і відповідно їх кількість)

	> список глядачів користувача (і відповідно їх кількість)

	> список людей, які лайкнули пост (кількість лайків)

  > список людей, які прокоментували пост (їх кількість)

  > текст, який було написано в коментарі

  > загальну інформацію про користувача, яка відображається на його профілі

  > локацію посту, якщо вона була вказана

Для того, щоб отримати данні про профіль потрібно мати логін і пароль, які знає
тільки власник профілю, або ID номер сторінки в соціальній мережі, який є в
загальному доступі.

Для того, щоб отримати данні про конкретну фотографію на сторінці необхідно
знати ID цього посту, який також є в загальному доступі.
